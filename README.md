# js-fullstack-starter

基于 React, Beidou 等技术的 JS 全栈开发项目模板

**⚠️开发中...**

## 特性

- ✔︎ 基于 beidou 框架, 高性能
- ✔︎ 同时支持 SSR 和 CSR
- ✔︎ React 框架
- ✔︎ 支持 ES6/ES7/ES8 现代语法
- ✔︎ 支持 SCSS/LESS, 自动补充浏览器前缀
- ✔︎ 支持单元测试
- ✔︎ 支持端到端测试
- ✔︎ 支持 Redux
- ✔︎ 支持热重载
- ✔︎ 支持 Egg.js 插件
- ✔︎ 支持多语言配置
- ✔︎ 集成了免费的性能分析工具 - alinode
